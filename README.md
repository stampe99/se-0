# se-0
A simple variant of neofetch written in Python.

# Installation

### Install python [here](https://python.org/)

git clone https://github.com/stampe99/se-0/ && cd se-0

```ps
pip install colorama
pip install sys
python fetch.py
```
### Required modules: 
- colorama
- sys
- ctypes
- platform
- psutil
- os
- distro
